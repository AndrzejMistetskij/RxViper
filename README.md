Download
--------

Grab via Maven:

```xml
<dependency>
    <groupId>com.dzaitsev.rxviper</groupId>
    <artifactId>rxviper</artifactId>
    <version>0.10.0</version>
</dependency>
```

Gradle:

```groovy
compile 'com.dzaitsev.rxviper:rxviper:0.10.0'
```
